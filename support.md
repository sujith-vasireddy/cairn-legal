# Support

Cairn is built by one person. If something's broken or 
confusing, I want to hear about it.

## Email

**Cairn_Support@proton.me**

(Replace with whatever support email you actually want to use 
— same one you put in privacy.md and terms.md for consistency)

I read every message.

## Reporting a bug

Include what you can:

- Your iPhone model (Settings → General → About → Model Name)
- Your iOS version (Settings → General → About → Software 
  Version)
- What you were doing when the issue happened
- Screenshots if possible

## Feature requests

I'm always interested in how you're using Cairn and what you 
wish it did differently. Email me — the smaller and more 
specific the feedback, the better.

## Response time

I try to respond within a few days. If you haven't heard back 
in a week, assume your email got lost and send it again.

## Data and privacy

For questions about what data Cairn collects, see the 
[privacy policy](https://sujith-vasireddy.github.io/cairn-legal/privacy.html).

For data deletion: you can delete everything Cairn has about 
you directly in the app under Settings → Privacy → Delete all 
my data. No email required.

Thanks for using Cairn.
